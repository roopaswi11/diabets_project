# 🩺 Diabetes AI Report Analyzer

## 📌 Project Introduction
Diabetes AI Report Analyzer is an AI-powered healthcare web application that analyzes diabetes lab reports using OCR and intelligent data processing. The system extracts sugar values from medical reports and automatically predicts diabetes status while providing personalized diet and meal recommendations.

---

# 🚀 Technologies Used
- Python
- Flask
- OpenCV
- Tesseract OCR
- HTML, CSS
- Matplotlib
- pdf2image
- python-docx

---

# ✨ Features
- Upload reports in PDF, PNG, JPG, and DOCX formats
- OCR-based text extraction
- Automatic sugar value detection
- Diabetes prediction system
- Diet recommendation
- Personalized meal plan generation
- Doctor alert system
- Diabetes chatbot
- Dashboard visualization

# ▶️ How to Run the Project

## 1️⃣ Install Required Libraries

```bash
pip install -r requirements.txt
```

## 2️⃣ Install Tesseract OCR

Download and install Tesseract OCR:
https://github.com/UB-Mannheim/tesseract/wiki

---

## 3️⃣ Run the Flask Application

```bash
python app.py
```

---

## 4️⃣ Open in Browser

```bash
http://127.0.0.1:5000
```

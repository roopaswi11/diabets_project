# 🩺 Diabetes AI Diet Recommendation System

## 📌 Project Description
This is a web-based application that analyzes diabetes reports and provides personalized diet recommendations.

Users can upload medical reports (PDF/Image/DOCX), and the system extracts sugar values using OCR and generates a diet plan using AI/logic.

---

## 🎯 Features
- 📤 Upload sugar reports (PDF, Image, DOCX)
- 🔍 Extract data using OCR
- 📊 Analyze blood sugar levels (Fasting, PP, HbA1c)
- 🥗 Generate diet recommendations
- 🚨 Doctor alert for critical cases
- 💬 Chatbot support (optional)
